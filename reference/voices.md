# 音色参考

## 可用音色

通义千问TTS（qwen3-tts-flash）支持以下音色：

| 音色ID | 性别 | 风格 | 推荐场景 |
|--------|------|------|---------|
| `Cherry` | 女 | 温柔甜美 | 播客主持、情感内容 |
| `Serena` | 女 | 知性优雅 | 新闻播报、专业讲解 |
| `Chelsie` | 女 | 活泼可爱 | 娱乐节目、年轻受众 |
| `Ethan` | 男 | 成熟稳重 | 专业讲解、深度对话 |

## 推荐搭配

### 双人播客

| 场景 | 女声 | 男声 |
|-----|------|------|
| 轻松对谈 | Cherry | Ethan |
| 专业访谈 | Serena | Ethan |
| 娱乐节目 | Chelsie | Ethan |

### 多人播客

```python
voices = {
    "host": "Cherry",      # 主持人
    "guest1": "Ethan",     # 嘉宾1
    "guest2": "Serena",    # 嘉宾2
    "narrator": "Serena",  # 旁白
}
```

## 语种支持

| 语种 | 参数值 |
|-----|-------|
| 自动识别 | `Auto` |
| 中文 | `Chinese` |
| 英文 | `English` |
| 日语 | `Japanese` |
| 韩语 | `Korean` |
| 法语 | `French` |
| 德语 | `German` |
| 西班牙语 | `Spanish` |
| 意大利语 | `Italian` |
| 葡萄牙语 | `Portuguese` |
| 俄语 | `Russian` |

## 使用建议

1. **单一语种**：如果文本是单一语种，明确指定 `language_type` 可显著提升质量
2. **混合语种**：中英混合文本建议使用 `Auto`
3. **情感表达**：通过标点符号和语气词增强情感表达
4. **文本长度**：单次合成最长 600 字符，超长文本需分段
